## BACKEND — Django APIs

Create a new Django app called `content` (if not already existing).

### 1. Smart Chat with AI (گفتگوی هوشمند)
POST /api/content/chat/
- Request: { conversation_id (optional), message, workspace_id }
- If no conversation_id, create new AIConversation
- Append user message to messages history
- Send full history to OpenAI gpt-4o with system prompt: "You are an expert Persian content creator assistant. Help users create high-quality content for social media, websites, and marketing channels."
- Append AI response to history, save to DB
- Return: { conversation_id, reply, messages_history }

GET /api/content/chat/ — list user's conversations (for this workspace)
GET /api/content/chat/{id}/ — get full conversation with messages
DELETE /api/content/chat/{id}/ — soft delete

### 2. Text Generation Tools
All endpoints below follow this pattern:
- POST request with { prompt, workspace_id, extra_params }
- Call OpenAI gpt-4o with appropriate system prompt
- Save result as Content object in DB
- Return { content_id, result_text }

Endpoints:
- POST /api/content/generate/text/ — general text generation
  - params: { topic, tone (formal/casual/friendly), length (short/medium/long), language (default: Persian) }
  
- POST /api/content/generate/scenario/ — content scenario/script
  - params: { topic, platform (telegram/instagram/website), goal }
  
- POST /api/content/generate/rewrite/ — rewrite existing text
  - params: { original_text, tone, instructions }
  
- POST /api/content/generate/summary/ — summarize text
  - params: { original_text, length (brief/detailed) }
  
- POST /api/content/generate/title/ — generate titles/headlines
  - params: { topic, count (default 5), style }
  
- POST /api/content/generate/cta/ — generate Call-To-Action
  - params: { product_or_service, goal, platform }
  
- POST /api/content/generate/hashtag/ — generate hashtags
  - params: { topic, platform, count (default 10), language }
  
- POST /api/content/generate/idea/ — generate content ideas
  - params: { niche, count (default 5), platform }

### 3. Image Generation
POST /api/content/generate/image/
- params: { prompt, size (1024x1024 / 1792x1024 / 1024x1792), style (vivid/natural), workspace_id }
- Call DALL-E 3 via OpenAI API
- Save result to GeneratedImage model
- Return { image_id, image_url, prompt_used }

GET /api/content/images/ — list workspace images (paginated)
DELETE /api/content/images/{id}/ — soft delete

### 4. Content Management (CRUD)
GET /api/content/ — list all content for workspace (filter by status, content_type, search by title)
POST /api/content/ — create content manually
GET /api/content/{id}/ — get single content with version history
PUT /api/content/{id}/ — update content (auto-create new ContentVersion before saving)
DELETE /api/content/{id}/ — soft delete (set is_active=False)

POST /api/content/{id}/archive/ — set status to archived
POST /api/content/{id}/restore/ — restore from archived

GET /api/content/{id}/versions/ — list all versions
POST /api/content/{id}/versions/{version_id}/restore/ — restore a specific version as current

### 5. Search
GET /api/content/search/?q=keyword&workspace_id=xxx
- Search in title and body fields
- Filter by content_type and status
- Return paginated results
